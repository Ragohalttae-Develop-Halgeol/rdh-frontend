# 윙커톤 3팀 : 방울

## 👋프로젝트 소개
국민대학교 동아리 WINK 와 더컴퍼스가 진행한 페르소나 웹 해커톤에 참여한 프로젝트입니다. (2024-08-31 ~ 2024-09-01)
이 해커톤에서는 특정 인물에 대한 페르소나를 분석하고, 그들이 필요로 하는 서비스를 창의적으로 기획하여 개발해보는 시간을 가졌습니다.
예를 들어, 인물의 나이, 성별, 취미/관심사, 일상 등을 고려해야 합니다.
